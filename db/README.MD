# Database Migration Tool

이 폴더는 로컬의 공정 라벨링 데이터를 Aiven 클라우드 데이터베이스로 마이그레이션하기 위한 스크립트를 포함합니다.

## 주요 기능
* **마스터 데이터 등록**: `fab` 및 `production_line` 테이블의 초기 정보를 설정합니다.
* **장비 정보 동기화**: JSON 메타데이터를 기반으로 장비를 각 라인에 할당하고 `device` 테이블에 등록합니다.
* **센서 데이터 일괄 적재**: `sensor_reading` 테이블에 대량의 센서 데이터를 500개 단위(Batch)로 신속하게 적재합니다.

## 실행 환경 및 설정
* **언어**: Python 3.x
* **필수 라이브러리**: `pymysql` (`pip install pymysql`)
* **인증**: SSL 접속을 위해 같은 폴더 내에 `ca.pem` 파일이 필요합니다.

## 사용 방법
1. `./02.라벨링데이터` 폴더에 마이그레이션할 JSON 파일들이 있는지 확인합니다.
2. `migration.py` 파일의 `db_config` 정보를 확인합니다.
3. 아래 명령어를 실행합니다:
   python migration.py
